# Srinath Bhuvanapalli - Personal Portfolio

This repository contains the digital headquarters of **Srinath Bhu** — a Content Strategist, Organic Growth Specialist, Digital Storyteller, Video Producer, and Creator.

## Overview
This is a premium, modern, editorial-style portfolio website designed to communicate authority, creativity, and strategic thinking. It is built using a dark-mode editorial layout inspired by leading tech design systems, featuring clean typography, strong visual hierarchy, and an evergreen philosophy ("Clarity Over Noise").

## Tech Stack
- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- Motion.dev (for animations)
- Font Awesome

## Features
- Fully Responsive Design
- Editorial-style Dark Mode
- Animated Timeline for Experience
- Interactive Project and Portfolio Cards
- Sticky Navigation with Scroll Progress Indicator
- SEO Optimized

## Local Development
To run this project locally, you can use any standard local web server (e.g., Live Server in VS Code, or Python's `http.server`):
```bash
python -m http.server 8000
```
Then visit `http://localhost:8000` in your browser.
